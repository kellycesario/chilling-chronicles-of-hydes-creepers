# Functional and Non-Funcional Requirements

## Functional Requirements:

1. The blog must have an easy-to-use interface for finding and reading "Creepers" book reviews
2. The blog must have a search bar that allows users to find specific reviews
3. The blog must have social media sharing buttons for each post

## Non-Functional Requirements:

1. Usability: The blog should be designed to be easy to navigate and use.
2. Performance: The blog should be optimized for fast load times to ensure users have a seamless experience.
3. Accessibility: The blog should be accessible to users with disabilities, following accessibility standards.
4. Availability: The blog should be available at all times to ensure users can access it when needed.
5. Mobile Responsiveness: The blog should be designed to work well on mobile devices, allowing users to access it from anywhere.
