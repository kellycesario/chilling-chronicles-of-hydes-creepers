# Functional and Non-Funcional Requirements

## Functional Requirements:

The blog must have an easy-to-use interface for finding and reading "Creepers" book reviews
The blog must have a search bar that allows users to find specific reviews
The blog must have social media sharing buttons for each post
The system must offer the ability to filter reviews by book genre, author, or reviewer

## Non-Functional Requirements:

Usability: The blog should be designed to be easy to navigate and use.
Performance: The blog should be optimized for fast load times to ensure users have a seamless experience.
Accessibility: The blog should be accessible to users with disabilities, following accessibility standards.
Availability: The blog should be available at all times to ensure users can access it when needed.
Mobile Responsiveness: The blog should be designed to work well on mobile devices, allowing users to access it from anywhere.
